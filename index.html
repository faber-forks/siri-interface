Web based Siri interface

This is a proof of concept for a web based Siri interface. This poc relies on some experimental browser api's (web-speech-api) which is currently only supported by Chrome. Make sure you allow the browser to access your microphone. The speech-api stops listening after 60 seconds, this is a build in limitation.

Try saying some thing to Siri, it's already listening! It may take some time for Siri to recognise or respond to your question, everything you say is send to google servers to be analysed.

Some of the commands:

  • My name is {{your name}}
  • Are you on twitter?
  • Where are you from
